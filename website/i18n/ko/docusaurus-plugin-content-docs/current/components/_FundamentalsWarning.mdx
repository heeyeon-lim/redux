:::경고

이 튜토리얼은 Redux의 원리와 개념을 설명하기 위해 **현재 Redux로 앱을 빌드하는 데 적합한 접근 방식으로 가르치는 Redux Toolkit을 사용하는 "최신 Redux" 패턴보다 더 많은 코드를 필요로 하는 이전 스타일의 Redux 로직 패턴을 의도적으로 보여드립니다**. 프로덕션에 바로 사용할 수 있는 프로젝트가 아닙니다.

Redux Toolkit으로 "최신 Redux"를 사용하는 방법을 알아보려면 이 페이지를 참조하세요:
